## Descrição



Automação de dois cénarios simples no site da https://easyjur.com, integrado a um CI no github actions e sonarQube.

## Project setup

```bash
$ npn install
```

## Run tests
```bash
# Execução assistida
$ npm run headed

# Execução em modo debug
$ npm run debug

# Execução em segundo plano
$ npm run test
```

## Stack utilizada

**Automação:** Node, TypeScript, Playwright
